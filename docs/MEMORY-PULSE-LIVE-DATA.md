# Memory and Pulse Live Data

Command Center v5 should replace mock memory, timeline, and metric data with Mobius Memory and Mobius Pulse when backend integration begins.

## Mobius Memory Feeds

- Organizational timeline
- Founder approvals
- Repository updates
- Voice intelligence events
- Architecture decisions
- Platform Change Proposals
- Decision history

## Mobius Pulse Feeds

- OIES
- MIS
- VCI
- Decision latency
- Memory reuse rate
- Department health
- Repository health trends
- Learning velocity

## UI Rule

Every recommendation shown in Command Center should include source, confidence, evidence, and memory reference.
