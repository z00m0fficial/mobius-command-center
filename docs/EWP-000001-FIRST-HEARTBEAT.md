# EWP-000001: First Heartbeat

## Mission

Display the first working Mobius Intelligence Loop in Command Center.

## Flow

Founder Request -> API -> Event Bus -> Atlas -> Memory -> Router -> Provider -> Memory Event -> Pulse -> MCMS -> Dashboard Refresh

## UI Requirements

- Founder request panel
- Live timeline
- Repository health
- Event stream
- Memory event count
- Pulse metric cards
- MCMS sync status

## Acceptance

The dashboard should clearly show whether the Intelligence Loop completed or failed.
