# Memory and Pulse Dashboard Integration

Mobius Command Center displays outputs from Mobius Memory and Mobius Pulse.

## Memory Views

- Organizational timeline
- Memory search
- Decision history
- Voice meeting records
- Architecture history
- Repository update history

## Pulse Views

- OIES
- MIS
- VCI
- Decision latency
- Memory reuse rate
- Department health
- Repository health trends
- Learning velocity

## Rule

Command Center displays intelligence. It does not own the source of truth for memory or metrics.
