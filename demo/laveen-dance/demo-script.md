# Laveen Dance Demo Script

## Demo Goal

Show how Mobius helps Laveen Dance operate with better memory, staffing visibility, instructor reliability tracking, and executive-level recommendations.

## Opening

"Mobius is not just an AI assistant. It is an Organizational Intelligence Operating System that helps a studio remember, coordinate, measure, and improve."

## Demo Prompt

"Summarize Laveen Dance operations this week, identify staffing risks, and recommend the next best action."

## Expected Demo Flow

1. Runtime creates a loop ID.
2. Brain checks organizational memory first.
3. Memory returns studio-relevant records.
4. Brain identifies staffing risk and operational priorities.
5. Dashboard displays telemetry.
6. Mobius recommends a next action.
7. Memory records the outcome.

## Dashboard Story

- Active Instructors: Shows staffing capacity.
- Class Coverage Risk: Flags uncovered or risky classes.
- Instructor Reliability Score: Tracks instructor accountability and recovery.
- Substitute Needs: Shows where backup instructors may be required.
- Parent Feedback: Captures lower-weight customer sentiment.
- Studio Follow-Ups: Shows sponsor, staff, parent, or scheduling actions.
- Recommended Actions: Gives leadership the next best move.

## Closing Line

"The value of Mobius is that Laveen Dance does not have to rely on scattered messages, memory, and spreadsheets. The organization starts learning from every class, every instructor interaction, every cancellation, and every operational decision."
