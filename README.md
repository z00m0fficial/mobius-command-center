# Mobius Command Center

Executive operating environment for the Mobius Organizational Intelligence Platform.

## Mission

Show leaders what is happening now, what needs attention, what changed, and what Mobius recommends next.

## Core Views

- Executive Overview
- Department Operations
- AI Coordination
- Repository and MCMS Health
- Organizational Memory Timeline
- OIES, VCI, and MIS Performance
- Platform Change Proposals
- Live Routing Decisions
- Predictive Recommendations

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
